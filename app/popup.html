Zen Alert - User Manual

All features of this extension are centered around the Views page. The alert system will not be active in until the first time you visit the Views page. After that, you will receve alerts until you close Chrome.

The “Assigned” view shows views created by your organization and pushed to you as an agent. These views cannot be modified or removed and you are typically restricted to 12 at a time. This extension allows for better management and custom use of these views.

The "Personal" folder contains views you have created. When selected you can edit these views by selecting the header called “Manage Your Views”. You can also rename these views quickly using the features outlined in this manual.

The "Watching" folder Allows you to focus your attention on only the Views you are interested in following. These Views will also alert you when a new ticket is submitted using a visual popup shown in your operating system notification bar.

Any View can be renamed by selecting a View and clicking the title on the right-hand panel. This will not modify the view for anyone but yourself. If you need to restore the original title simply backspace the one you entered. If a name is too long it may be truncated in the View list.

Select the checkbox next to a View to add it to the watch list. There is no restriction on the number of items added to the watch list, and you can add them from both the Assigned and Personal groups simultaneously.

View links work exactly as they did and the unmodified version of the page. The only exception being they can be renamed.

Ticket counters are very similar to their unmodified counterparts. Gray indicates no active tickets for the View, Green indicates 5 or fewer, Yellow indicates 6 - 10 tickets, and Red indicates over 10.

Any View added to the watching list is capable of producing alerts which will be sent to the system notification tray. These notifications may look different based on the operating system you're currently using and will include the title of the View that created the alert.
